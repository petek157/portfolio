Indianhead Valley Condos
  • Web App
  • iOS
  • IoT
  • Stripe
  • Twilio

Indianhead Valley's frontend website using PHP and javscript / jQuery to back a custom online reservation system where guest can do the usual price and availablity checking and book units. Payment is done via the Stripe Payments API. The database is base on the ParsePlatform using MongoDB. The phone and messaging system is run through the Twilio API.

The admin side allows managers to perform all CRUD functions of a Resevation Managment System, including, units, guests, reservation, payments, financials and reservation/guest notes. Also core is a 13 unit reservation calander.

The iOS app includes all of the functionality of the backend of the website but also includes a few IoT objects. Temperature sensors that can be viewed in the app and alert managment to low temperatures. Outside lights when there are no guests or when guests want an unpoluted view of the stars can be remotely shut off.
