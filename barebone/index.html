Choose your version.

The RoR version allows people visiting the site to create a user and comment on how the site could be improved or what they think about projects in the portfolio.
The bare bone version is only HTML, CSS, JS and a JSON data file. Simple, quick and clean. Sites dont have to be overly developed to be effective.
The React version with AWS Amplify showcases a few of the many possibilities of connecting to AWS.
You'll notice that not one of the site versions is styled very well. There is a very good reason for that, I have no style. Seeing something that I like, duplicating it is no problem. But seen as this site is supposed to show what "I" can do, having it look pretty based on someone elses design seems a bit unauthentic.
